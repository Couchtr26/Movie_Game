# 🎬 Movie Game – Full Stack Demo App

Build and simulate custom movies from scripts, producers, actors, effects, and scandals.  
This app showcases full-stack capabilities using **React + FastAPI + SQLite**, fully containerized with Docker.

---

## ✨ Features

- 🎥 Create movies by combining scripts, producers, actors, effects, and scandals  
- 🧠 Backend powered by FastAPI (Python)  
- 🌐 Frontend built with React + TailwindCSS  
- 🗂 Uses a real SQLite game database  
- 🐳 Dockerized for easy deployment and quick demos  

---
